切片是引用¶

切片在内存中使用的是引用机制¶

In [84]:
a = np.array([0,1,2,3,4])
b = a[2:4]
b
Out[84]:
array([2, 3])
In [85]:
b[0]=10
a
Out[85]:
array([ 0,  1, 10,  3,  4])

引用机制意味着,Python并没有为b分配新的空间来存储他的值，而是让b指向了a所分配的内存空间，因此，改变b会改变a的值：¶

而这种现象在列表中并不会出现¶

In [86]:
a = [1,2,3,4,5]
b = a[2:4]
b[0]=10
print(a)

[1, 2, 3, 4, 5]

这样做的好处在于，对于很大的数组，不用大量复制多余的值，节约了空间；¶

缺点在于，可能改变一个值改变另一个值得情况¶

一个解决方法是使用copy()方法产生一个复制，这个复制会申请新的内存¶

In [88]:
a = np.array([0,1,2,3,4])
b = a[2:4].copy()
b[0]=10
a
Out[88]:
array([0, 1, 2, 3, 4])

数组连接¶

有时需要将不同的数组按照一定的顺序连接起来：

concatenate((a0,a1,...,aN),axis=0) axis代表拼接方向，默认是0，即按列拼接，若果为1，就是按行拼接

注意：这些数组要用（）包括到一个元祖中去。¶

除了给定的轴外，这些数组其他轴的长度必须是一样的¶

In [48]:
x = np.array([[0,1,2],[10,11,12]])
y = np.array([[50,51,52],[60,61,62]])
print(x.shape)
print(y.shape)

(2, 3)
(2, 3)

默认沿着第一维进行连接¶

In [50]:
z = np.concatenate((x,y))
z
Out[50]:
array([[ 0,  1,  2],
       [10, 11, 12],
       [50, 51, 52],
       [60, 61, 62]])

沿着第二维进行连接¶

In [51]:
z = np.concatenate((x,y),axis=1)
z
Out[51]:
array([[ 0,  1,  2, 50, 51, 52],
       [10, 11, 12, 60, 61, 62]])

注意到这里x和y的形状是一样的，还可以将他们连接成三维的数组，但是concatenate不能够提供这样的功能，不过可以这样：¶

In [52]:
z = np.array((x,y))
z
Out[52]:
array([[[ 0,  1,  2],
        [10, 11, 12]],

       [[50, 51, 52],
        [60, 61, 62]]])

事实上，Numpy提供了分别对应这三种情况的函数：¶

  • vstack
  • hstack
  • dstack
In [53]:
np.vstack((x,y)) ## 垂直堆叠
Out[53]:
array([[ 0,  1,  2],
       [10, 11, 12],
       [50, 51, 52],
       [60, 61, 62]])
In [54]:
np.hstack((x,y))  ## 横向堆叠
Out[54]:
array([[ 0,  1,  2, 50, 51, 52],
       [10, 11, 12, 60, 61, 62]])
In [56]:
np.dstack((x,y))  ##三维堆叠
Out[56]:
array([[[ 0, 50],
        [ 1, 51],
        [ 2, 52]],

       [[10, 60],
        [11, 61],
        [12, 62]]])
